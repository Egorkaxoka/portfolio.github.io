🚀 Портфолио | Егор Темников — Fullstack разработчик

> Современное, адаптивное и анимированное персональное портфолио с тёмной/светлой темой, интерактивными частицами и плавными переходами.

✨ Возможности

🌗 **Переключение тем** — тёмная и светлая темы с сохранением выбора в `localStorage`
🎆 **Интерактивные частицы** — canvas-анимация, реагирующая на движение мыши
⌨️ **Typing-эффект** — динамическая смена ролей в hero-секции
🎬 **Reveal-анимации** — плавное появление элементов при скролле через `IntersectionObserver`
🃏 **3D Tilt-эффект** — наклон карточек при наведении (только на десктопе)
📊 **Анимированные счётчики** — плавный подсчёт статистики
📱 **Полная адаптивность** — мобильное бургер-меню, оптимизированная сетка
⚡ **Оптимизация производительности**:
  - Остановка анимации при неактивной вкладке
  - Lazy loading для изображений
  - Уважение `prefers-reduced-motion`
  - Адаптивное количество частиц для мобильных устройств
  - 🔝 **Кнопка «Наверх»** с плавным скроллом
🎯 **Активная навигация** — автоматическая подсветка текущей секции
